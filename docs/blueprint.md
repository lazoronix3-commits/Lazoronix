# **App Name**: Lazoronix Recovery

## Core Features:

- Secure Assessment Submission: A secure, encrypted web form for users to submit initial case details for a confidential forensic recovery assessment.
- AI-Powered Assessment Preparation Tool: An interactive AI tool that guides users through a series of questions to help them gather and understand what critical information is relevant for their specific digital asset recovery scenario, before submission.
- Recovery Specialist Consultation Request: A dedicated interface for users to request a confidential consultation directly with a recovery specialist.
- Informative Content Display: Structured sections to clearly present services, expertise, 4-step recovery methodology, 'why choose us' differentiators, and frequently asked questions.
- Trust & Security Affirmation Display: Prominent display of trust elements like 'Secure encrypted case submission', 'Confidential review', and 'Evidence-driven. Secure. Confidential.' statements throughout the site.

## Style Guidelines:

- Color Palette: Dark scheme to convey professionalism, sophistication, and trust. Primary accent color: A rich, professional blue (#467BE0) symbolizing technology and reliability. Background color: A very dark, desaturated blue-grey (#1A1E24) providing a deep, stable foundation. Accent color: A vibrant, clear aqua (#4BC6EA) for highlights and call-to-action elements, offering effective contrast.
- Headlines: 'Space Grotesk' (sans-serif) for a modern, tech-informed and precise feel. Body Text: 'Inter' (sans-serif) for exceptional legibility, neutrality, and professionalism, suitable for explaining complex information.
- Utilize minimalist, sophisticated line icons. Employ icons that represent concepts like security (e.g., locks, shields), digital assets (e.g., blockchain links, crypto symbols), processes (e.g., sequential steps, arrows), and confidentiality (e.g., envelopes, privacy shields). All icons should maintain a consistent, high-end aesthetic.
- Employ a clean, structured, and spacious layout with ample negative space to enhance readability and a sense of premium quality. Content is organized into clear, digestible blocks and card-based sections for services and features. Strong visual hierarchy ensures key information, such as headlines and calls-to-action, stands out. Responsive design principles will be applied for optimal viewing across various devices.
- Implement subtle and purposeful animations to enhance user experience without distraction. These include smooth hover effects for interactive elements (buttons, links), gentle fade-in transitions for sections upon scroll, and discrete loading indicators, maintaining a professional and polished feel.
